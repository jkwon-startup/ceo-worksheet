# 1인 기업가 마케팅 검증 워크시트 (7종)

브라우저에서 바로 작성하는 웹 워크시트입니다. 작성 내용은 **브라우저에 자동 저장**되며, **마크다운(.md) / 텍스트(.txt) / PDF(인쇄)** 로 내려받을 수 있습니다.

## 특징

- **자동 저장**: 입력 즉시 브라우저(localStorage)에 저장 — 새로고침해도 유지
- **진행률 표시**: 워크시트별 완성 체크리스트로 `N/7 완료` 추적
- **다운로드**: 마크다운(표·체크박스 보존) · 텍스트 · 인쇄/PDF
- **설치 불필요**: 단일 HTML 파일, 빌드 단계 없음
